發表於 2020-03-06 更新於 2020-03-08 分類於 HTML

HTML (Hyper Text Markup Language)

  • 1993年，由Tim Berners Lee(英國電腦科學家，全球資訊網的發明者)，此規範自1996年開始，由W3C(全球資訊網協會) 維護至今。

  • HTML規範統一定義網際網路的傳輸文件格式，以利網際網路接收方與發送方都能輕易的讀取解析資訊。

  • HTML標籤大致分為兩種

    1. 成對出現：巢狀元素,需要在中間插入內容
    2. 單獨出現：空元素,個體本身代表內容

  • HTML在大多數的網頁瀏覽器中，並沒有強行制定接收後呈現的方式，故有時就算有錯誤(例如成對標籤沒加上尾巴，或是沒寫雙引號)，仍然能夠正常顯示。

  • Semantic HTML (語意化標籤) ，為加強文件的文字意涵，對一些裝置在讀取HTML文本時，可以更明確的知道哪些需要強調，哪些是重要的關鍵字

  • HTML於HTML5以前需要定義DTD(文件類型聲明)，因每一版本的定義都有些許差異，若果沒有定義，而使用網頁瀏覽器時，可能會出現不符合期望的結果(因使用的網頁瀏覽器或作業系統環境的差別，使用者可能彼此看到的畫面會不相同)，但是HTML5就沒有定義DTD。
