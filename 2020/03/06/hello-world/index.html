標籤 定義 描述
<font color="#dd0000"><!--...--></font> 註釋 會被瀏覽器忽略
<!DOCTYPE> 文檔類型 告知瀏覽器文檔所使用的 HTML 規範
<a> 超鏈結 透過href 屬性，指定點擊前往的網頁
<abbr> 縮寫 通過對縮寫詞語進行標記，能夠為瀏覽器、拚寫檢查程序、翻譯系統以及搜索引擎分度器提供有用的信息
<address> 地址 文檔作者或擁有者的聯絡訊息
<area> 圖像映射中的區域 定義圖像映射內部的區域（圖像映射指的是帶有可點擊區域的圖像）
<b> 粗體 用於強調，並表現粗體效果
<base> 頁面中所有鏈結的基準 URL 為頁面上的所有鏈結規定默認地址或默認目標
<bdo> 文本顯示的方向 標簽覆蓋默認的文本方向
<blockquote> 長的引用 定義摘自一段說明所使用的引用區塊
<body> 文檔的主體 文檔包含的所有內容
<br> 換行符 插入換行符
<button> 按鈕 定義按鈕，也可以在該元素內放置內容，比如文本或圖像
<caption> 表格標題 定義表格的標題，每個表格只能規定一個標題
<cite> 引用 對參考文獻的引用進行定義
<code> 程式碼 表現出電腦程式碼的片段
<col> 表格列的屬性 為表格中的一個或多個列定義屬性值
<colgroup> 表格列的分組 用於對表格中的列進行組合，以便對其進行格式化
<dd> 定義的描述 定義一個定義列表中對項目的描述
<del> 刪除字段 定義文檔中已刪除的字段
<dfn> 定義項目 定義一個定義項目
<div> 文檔中的一個部分 常用於組合塊級元素，以便通過樣式表來對這些元素進行格式化
<dl> 定義列表 用於結合
（定義列表中的項目）和
（描述列表中的項目）
<dt> 定義的項目 定義列表中的項目
<em> 強調文本 呈現為被強調的文本
<fieldset> 元素組合 用於從邏輯上將表單中的元素組合起來，會在相關表單元素周圍繪製邊框
<form> 表單 用於創建供用戶輸入的 HTML 表單
<h1> to <h6> 定義標題 1 到標題 6

定義最小的標題
<head> 關於文檔的信息 可以包含腳本、指引瀏覽器找到樣式表、提供元信息
<hr> 水平線 插入水平線
<html> 宣告檔案類型 告知瀏覽器這是一個 HTML 文檔
<i> 斜體 呈現斜體的文本
<iframe> 行內的子窗口（框架) 創建包含另一個文檔的行內框架
<img> 圖像 定義 HTML 頁面中的圖像
<input> 輸入控制元件 根據不同的Type屬性，輸入元件有多種形態
<ins> 插入文本 定義文檔的其餘部分之外的插入文本
<kbd> 鍵盤文本 表示文本是從鍵盤上鍵入的
<label> 表單控制元件的標註 為 input 元素定義標簽（label）
<legend> 定義標題 legend 元素為以下元素定義標題（caption）：
、
<li> 列表的項目 定義列表項
<link> 資源引用 定義文檔與外部資源之間的關係
<map> 圖像映射 用於定義客戶端圖像映射。(圖像映射指的是帶有可點擊區域的圖像)
<menu> 菜單列表 定義菜單列表
<meta> 元信息 可提供有關頁面的元信息（meta-information），比如針對搜索引擎和更新頻度的描述和關鍵詞
<noscript> 腳本未被執行時的替代內容 用來定義在腳本未被執行時的替代內容（文本）
<object> 嵌入對象 定義一個嵌入的對象。請使用此元素向您的 XHTML 頁麵添加多媒體。此元素運行您規定插入 HTML 文檔中的對象的數據和參數，以及可用來顯示和操作數據的代碼
<ol> 有序列表 定義有序列表

標籤 定義 描述
<article> 獨立的內容 表示文檔、頁面、應用或網站中的獨立結構
<aside> 頁面內容之外的內容 表示一個和其餘頁面內容幾乎無關的部分，被認為是獨立於該內容的一部分並且可以被單獨的拆分出來而不會使整體受影響
<audio> 聲音內容 定義聲音，比如音樂或其他音頻流
<bdi> 隔離 設置一段文本，使其脫離其父元素的文本方向設置
<canvas> 圖形 只是圖形容器，仍然必須使用腳本來繪製圖形
<command> 命令按鈕 表示用戶能夠調用的命令
<datalist> 下拉列表 定義選項列表。請與 input 元素配合使用該元素，來定義 input 可能的值
<details> 元素的細節 用於描述文檔或文檔某個部分的細節
<embed> 外部交互內容或插件 定義嵌入的內容，比如插件
<figcaption> figure標題 定義 figure 元素的標題
<figure> 媒介內容的分組，以及它們的標題 規定獨立的流內容（圖像、圖表、照片、代碼等等）
<footer> 頁尾 定義 section 或 page 的頁尾
<header> 頁首 定義 section 或 page 的頁首
<hgroup> 有關文檔中的 section 的信息 用於對網頁或區段（section）的標題進行組合
<keygen> 表單密鑰 用於表單的密鑰對生成器字段
<mark> 有記號的文本 定義帶有記號的文本
<meter> 預定義範圍內的度量 定義度量衡。僅用於已知最大和最小值的度量
<nav> 導覽列 定義導覽鏈結的部分
