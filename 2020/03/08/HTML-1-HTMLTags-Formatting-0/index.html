HTML標籤-格式

發表於 2020-03-08
標籤 定義 描述
<abbr> 縮寫 通過對縮寫詞語進行標記，能夠為瀏覽器、拚寫檢查程序、翻譯系統以及搜索引擎分度器提供有用的信息
<address> 地址 文檔作者或擁有者的聯絡訊息
<b> 粗體 用於強調，並表現粗體效果
<bdo> 文本顯示的方向 標簽覆蓋默認的文本方向
<blockquote> 長的引用 定義摘自一段說明所使用的引用區塊
<cite> 引用 對參考文獻的引用進行定義
<code> 程式碼 表現出電腦程式碼的片段
<del> 刪除字段 定義文檔中已刪除的字段
<dfn> 定義項目 定義一個定義項目
<em> 強調文本 呈現為被強調的文本
<i> 斜體 呈現斜體的文本
<ins> 插入文本 定義文檔的其餘部分之外的插入文本
<kbd> 鍵盤文本 表示文本是從鍵盤上鍵入的
<pre> 預格式化文本 被包圍在 pre 元素中的文本通常會保留空格和換行符。而文本也會呈現為等寬字體
<q> 短的引用 定義短的引用
<samp> 樣本計算機代碼 定義樣本文本
<small> 小型文本 將旁註 (side comments) 呈現為小型文本
<s> 刪除線 加刪除線的文本
<strong> 強調文本 重要的文本
<sup> 上標文本 定義上標文本
<sub> 下標文本 定義下標文本
<var> 變量 定義變量。可以與 pre 及 code 標簽配合使用

HTML5 新標籤

標籤 定義 描述
<bdi> 隔離 設置一段文本，使其脫離其父元素的文本方向設置
<mark> 有記號的文本 定義帶有記號的文本
<meter> 預定義範圍內的度量 定義度量衡。僅用於已知最大和最小值的度量
<progress> 任何類型的任務的進度 定義運行中的進度（進程）
<rp> 不支持 ruby 元素時的顯示內容 若瀏覽器不支持 ruby 元素顯示的內容
<rt> ruby 註釋 定義 ruby 註釋的解釋
<ruby> ruby 註釋 定義 ruby 註釋
<time> 日期/時間 定義時間和日期
<wbr> 換行 瀏覽器會判斷是否此處已經在最邊框，若已經在最邊框，有 wbr的話就會斷行，否則就不會

HTML 5 已不支持

標籤 定義 描述
<acronym> 首字母縮寫 通過對只取首字母縮寫進行標記，能夠為瀏覽器、拚寫檢查程序、翻譯系統以及搜索引擎分度器提供有用的信息
<big> 大號字體效果 其字體比周圍的文字要大一號
<center> 水平置中 對其包圍的文本和內容進行水平置中處理
<font> 文字設定 定義了該內容的字體大小、顏色與表現
<strike> 刪除線 定義加刪除線的文本
<tt> 打字機文本 用於給文本排版，使其等寬展示，就像電報那樣
<u> 下劃線 渲染為一個實線下劃線
