HTML-基礎

發表於 2020-03-11 分類於 HTML

這個章節會講解HTML的基礎標籤

  • <!DOCTYPE>- 為了讓瀏覽器知道這個頁面的文件類型是什麼，瀏覽器根據該標籤來決定要怎麼支援顯示出來的畫面，如果沒有宣告的時候，大部份的瀏覽器會以它們自訂的默認的 DOCTYPE 解釋該頁的 HTML，有可能會出現使用者看到的畫面不完全相同的狀況 <html></html>-除了宣告用的<!DOCTYPE>，所有要顯示的內容都要放在這個標籤的範圍內，它又被稱為根元素(root element) <head></head>-這裡是要放置不顯示於畫面上的預存資訊，像是引用的樣式文件、搜尋結果的關鍵字、頁面說明。 <body></body>-這裡是要放置顯示於畫面上的內容，像是文字、圖片、影片。 <meta charset="utf-8">-指定你的網頁文件要使用的字元編碼。 <title></title>-呈現於網頁瀏覽者眼前的網頁標題
