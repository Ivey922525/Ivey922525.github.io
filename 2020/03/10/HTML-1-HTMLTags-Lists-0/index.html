標籤 定義 描述
<ul> 無序列表 定義無序列表
<ol> 有序列表 定義有序列表
<li> 列表的項目 定義列表項
<dl> 定義列表 用於結合 dt （定義列表中的項目）和 dd （描述列表中的項目）
<dt> 定義的項目 定義列表中的項目
<dd> 定義的描述 定義一個定義列表中對項目的描述
<menu> 菜單列表 定義菜單列表
<menuitem> 彈出菜單調用的命令/菜單項目 定義用戶可以從彈出菜單調用的命令/菜單項目
